Three-Phase Soft Starter Control Code

This repository contains Arduino-based control code for a three-phase induction motor soft starter.

Overview: The system uses zero-cross detection and phase-angle control to gradually increase RMS voltage during startup, reducing inrush current.

Hardware: Microcontroller with at least 3 external interrupt pins (e.g. Arduino Mega)
